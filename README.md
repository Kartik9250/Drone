# Arduino based drone
An arduino based drone capable of stable flight.

## Requirements
	- Arduino
	- IMU: MPU6050
	- ESCs
	- BLDC motors
	- Radio Transmitter
	- Radio Receiver
	- Lipo Battery

## Installing Arduino IDE
Arduino ide could easily be installed by following [This link](https://www.arduino.cc/en/software)
